Markdown Cheatsheet (Comprehensive)

1. Headers

# H1
## H2
### H3
#### H4
##### H5
###### H6

Alternatively, for H1 and H2:
Heading 1
=========
Heading 2
---------

2. Emphasis

*Italics* or _Italics_
**Bold** or __Bold__
***Bold and Italics*** or ___Bold and Italics___
~~Strikethrough~~

3. Lists

Unordered Lists

* Item 1
* Item 2
  * Nested Item 2a
  * Nested Item 2b

(You can also use + or - for unordered lists)

Ordered Lists

1. First item
2. Second item
3. Third item

Task Lists

- [x] Completed task
- [ ] Incomplete task

4. Links

Inline Link: [OpenAI](https://openai.com)
Link with Title: [OpenAI](https://openai.com "OpenAI Homepage")
Reference Link: [OpenAI][1]
...later in the document...
[1]: https://openai.com

5. Images

Inline Image: ![Alt text](image-url.jpg)
Image with Title: ![Alt text](image-url.jpg "Image Title")
Linked Image: [![Alt text](image-url.jpg)](https://example.com)

6. Blockquotes

> This is a blockquote.
> It can span multiple lines.
>> And it can be nested!

7. Code

Inline Code: Use the `printf()` function.
Code Blocks:
```python
def hello_world():
    print("Hello World")
```

8. Tables

| Syntax | Description |
| ----------- | ----------- |
| Header | Title |
| Paragraph | Text |

Alignment:
| Left-aligned | Center-aligned | Right-aligned |
| :--- | :---: | ---: |

9. Horizontal Rules

You can use three or more asterisks, dashes, or underscores:

***
---
___

10. Escaping Characters

If you need to use a literal character that would otherwise trigger Markdown formatting, escape it with a backslash `\`.

\*This will not be italicized\*
