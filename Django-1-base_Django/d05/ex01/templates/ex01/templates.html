Template Engine

Django's template system separates design from Python code.

  • Blocks: Allow inheritance. You define a block in a base template and override it in child templates.
  • Loops: The {% verbatim %}{% for item in list %}{% endverbatim %} tag iterates over arrays.
  • Conditions: The {% verbatim %}{% if condition %}{% endverbatim %} tag evaluates logic.
  • Variables: Displayed using double curly braces: {% verbatim %}{{ variable_name }}{% endverbatim %}.
